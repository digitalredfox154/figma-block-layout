## Исправление CTA

Production-сборка использует файл `Rectangle240649630(1)_d16e0e57.png`. Он был загружен в каталог `public_html/manus-storage/` с правами `644`. Для совместимости со старым именем на сервере также создан файл `Rectangle240649630(1)_7db956e6.png` с тем же оригинальным содержимым.

## Контроль публикации

Прямые запросы к обоим URL возвращают настоящий PNG, а не HTML-заглушку. После последовательной прокрутки страницы для активации ленивой загрузки браузерный DOM-аудит показал `brokenImageCount: 0` при `imageCount: 57`.
